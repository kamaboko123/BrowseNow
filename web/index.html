BrowseNow!

表示中のページをTwitterに投稿できるブックマークレット

ブラウザのブックマークバーに登録しておけば2クリックで表示中のページをTwitterにシェアできます。

How to use

1. 以下のリンクをブラウザのブックマークバーにドラッグ・アンド・ドロップして、ブックマークとして登録します。*
BrowseNow!

2. 登録したブックマークをクリックして開きます。

3. Twitterの認証画面が開くので、連携を許可します。

4. 共有したいページでブックマークをクリックすると、Twitterにそのページでを共有できます

*ドラッグ・アンド・ドロップでブックマーク登録できないブラウザの場合

以下のURLをブックマークバーに登録してください。
